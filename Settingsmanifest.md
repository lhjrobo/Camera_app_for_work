# What should be in settings menu

- 1. enable/disable to ignore device orientation lock(force the app to rotate according to device orientation)
- 2. options for initial state of following:(default, fixed, or last used state)
    - folder
        - default: root folder, fixed: user selected folder, last used folder
    - shutter position (portrait/landscape)
        - default: centered, fixed: current shutter position, last used position
    - labeling mode
        - default: single, fixed: current labeling mode, last used mode
    - mode(photo or video)
        - default: photo, fixed: current mode, last used mode
    - flash mode
        - default: auto, fixed: current flash mode, last used mode
    - last used camera(front/rear)
        - default: front, fixed: current camera, last used camera

these options should be saved in app's local storage
